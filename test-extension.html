CoinTracker 扩展测试

安装步骤

  1. 打开 Chrome 浏览器
  2. 在地址栏输入 chrome://extensions/
  3. 打开右上角的"开发者模式"
  4. 点击"加载已解压的扩展程序"
  5. 选择项目的 dist 文件夹
  6. 扩展应该会出现在扩展列表中

测试步骤

  1. 点击浏览器工具栏中的 CoinTracker 图标
  2. 打开浏览器开发者工具（F12）
  3. 在 Console 标签中查看日志信息
  4. 检查是否有错误信息

调试信息

如果扩展一直显示"加载中"，请检查以下内容：

  • Service Worker: 在 chrome://extensions/ 页面检查背景脚本是否正常运行
  • 控制台错误: 查看popup和背景脚本的控制台是否有错误
  • 网络请求: 检查 Network 标签是否有API请求
  • 权限: 确认扩展有访问 api.coingecko.com 的权限
