# CoinTracker Chrome Extension

## 🚀 功能特性
- 实时加密货币价格监控
- K线图详情分析
- 搜索并添加任意币种
- 价格变化通知
- 本地数据存储

## 📈 已修复的问题
- **API频率限制 (429错误)**：增加了请求队列和缓存机制
- **智能缓存**：不同类型数据使用不同缓存时长
- **友好错误提示**：用户友好的中文错误信息
- **自动重试**：遇到限制时自动延迟重试

## 🛠️ 技术优化
1. **请求队列**：所有API请求排队执行，避免并发
2. **智能缓存**：
   - 价格数据：3分钟缓存
   - 搜索结果：15分钟缓存
   - 历史数据：10分钟缓存
   - 热门币种：30分钟缓存
3. **频率控制**：请求间隔1.5秒
4. **自动更新**：改为10分钟一次（原来5分钟）

## 🔧 安装步骤
1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `dist` 文件夹
6. 享受使用！

## 💡 使用提示
- 如遇到"API请求过于频繁"提示，请等待几分钟后再试
- 点击币种名称可查看K线图详情
- 数据会自动缓存，减少API调用
- 支持7天、30天、90天、1年的K线图

## 🔍 故障排除
如果还是遇到429错误：
1. 等待5-10分钟后重试
2. 重启浏览器刷新扩展
3. 检查网络连接状态
